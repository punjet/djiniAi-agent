## ПРОЕКТЫ

### AI Support Agent — компания по продаже автобусных билетов
*RAG + Python State Machine + Internal API + Docker*

RAG-система поддержки клиентов поверх базы знаний с правилами компании. Ключевая задача — разгрузить 10 операторов, сохранив качество обслуживания.

- **RAG по правилам компании:** Векторизация внутренней документации (политики возвратов, правила пересадок, регламенты обработки жалоб). Настройка retrieval с учётом специфики коротких запросов клиентов в чате.
- **Сложный prompt engineering для эскалации:** Агент должен сам решать, когда передать диалог живому оператору. Разработал многоуровневую логику: LLM оценивает тип проблемы, уровень неопределённости, эмоциональный тон и историю диалога — минимизация ложных escalations критична при ограниченном пуле из 10 операторов.
- **Python state machine:** Конечный автомат управлял стадиями диалога (ідентифікація → пошук → уточнення → вирішення → рішення про ескалацію → закриття). Каждое состояние определяло набор доступных инструментов и формат контекста для LLM.
- **Интеграция с внутренним API:** Tool Calling для получения в реальном времени: статус и детали бронирования, маршрут рейса (откуда/куда, остановки), текущее местоположение автобуса, актуальное расписание и задержки.
- **Деплой и инфраструктура:** Задеплоен в Docker-контейнере на Ubuntu VPS; docker-compose с PostgreSQL для персистентного хранения истории диалогов; автоматический restart при падении.
